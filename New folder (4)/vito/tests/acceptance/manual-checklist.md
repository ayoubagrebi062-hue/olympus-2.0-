# Manual UAC Checklist

> Complete this checklist before each release.
> Date: ___________
> Tester: ___________

## Visual Quality

- [ ] Homepage looks professional
- [ ] Colors are consistent throughout
- [ ] Fonts are readable at all sizes
- [ ] Images load properly
- [ ] No broken layouts or overlapping elements
- [ ] Dark mode works correctly (if applicable)

## Onboarding Experience

- [ ] Flow is intuitive and clear
- [ ] Progress indicator shows current step
- [ ] Animations are smooth and not jarring
- [ ] Can go back without losing data
- [ ] Error messages are helpful

## Build Quality

- [ ] Generated preview looks like a real website
- [ ] Content is relevant to the prompt
- [ ] Styling is applied (not raw HTML)
- [ ] No obvious placeholder text (Lorem Ipsum, etc.)
- [ ] Layout is responsive
- [ ] Would I show this to a client? (yes/no)

## Downloaded Code

- [ ] Unzip works without errors
- [ ] `npm install` succeeds without major warnings
- [ ] `npm run dev` starts the application
- [ ] App matches the preview
- [ ] Code is reasonably organized
- [ ] No obvious security issues in code

## Chat Refinement

- [ ] Changes are applied to the preview
- [ ] Previous work is preserved
- [ ] Response makes sense contextually
- [ ] Multiple refinements work correctly

## Performance

- [ ] Pages load in under 3 seconds
- [ ] No visible layout shift after load
- [ ] Buttons respond immediately to clicks
- [ ] Build progress updates smoothly
- [ ] No freezing or unresponsiveness

## Accessibility

- [ ] All interactive elements are keyboard accessible
- [ ] Color contrast is sufficient
- [ ] Screen reader can navigate the page
- [ ] Focus indicators are visible

## Overall

- [ ] I would use this product personally
- [ ] I would recommend this to a friend
- [ ] I encountered no frustrating bugs
- [ ] The experience felt polished
